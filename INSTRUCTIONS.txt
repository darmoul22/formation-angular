ENVIRONMENT SETTINGS :
1 - Install Node/npm (for angular 17 : NodeJs ^18.13.0 || ^20.9.0)
  Or
  - Install Nvm (guide : https://www.freecodecamp.org/news/node-version-manager-nvm-install-guide/)

2 - Install Angular Cli : npm install -g @angular/cli

TO START A NEW PROJECT :
 - ng new 'app-name' --no-standalone

TO GENERATE FILES AND SCHEMATICS
  - ng generate 'schematic'
  - for reference : https://angular.io/cli/generate



  new component : ng generate component 'component-name' (--no-standalone)
  new module : ng generate module 'module-name' --routing
  environments : ng g environments
  service : ng g service 'service-name'


to start json server :
  - npm install
  - json-server --watch db.json --port 3000
